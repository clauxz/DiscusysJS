Steps to use presentation template:
=========================

1. Copy the index.php and paste it with required name
2. Open the new file using any text editor.
3. Search for the line "<!-- Exported data begins here -->"
4. Copy paste the exported contents from slid.es between the section "<!-- Exported data begins here -->" and "<!-- Exported data ends here -->"
5. You can now access the presentation using url, www.williamwidjaja.com/presentation/<file>.php
    for master: add "?master" to the presentation url
	for joining the presentation: add "?join" to the presentation url
